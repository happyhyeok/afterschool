🥚 [5차시] 신비한 알과 점수 상자(변수)

1. 도입: 알을 깨워서 키워보자! 🐣

안녕 친구들! 오늘은 세 가지 신비한 알을 만나볼 거예요.
바로 불의 알, 이끼의 알, 물의 알이에요!
이 알들은 우리가 정성껏 클릭해줘야만 멋지게 깨어날 수 있답니다.

📌 오늘의 목표: '변수'를 만들어 클릭할 때마다 경험치를 쌓고, 알을 진화시켜 봅시다!

2. 전개: 변수는 신비한 요술 상자! 📦

게임을 만들 때 점수, 남은 시간, 내 이름 등을 기억해둬야 할 때가 있죠?
이렇게 데이터를 저장하는 '이름표가 붙은 요술 상자'를 우리는 변수라고 불러요.

변수 상자 안에는 숫자나 글자를 담을 수 있어요. 하지만 아주 중요한 마법의 규칙이 하나 있어요!

🚨 변수의 중요 규칙: 한 번에 오직 한 개만!

변수 상자에는 한 번에 딱 한 개의 데이터만 담을 수 있어요.
만약 '5'가 들어있는 상자에 새롭게 '10'을 넣으면 어떻게 될까요?
상자가 꽉 차서 원래 있던 '5'는 뿅! 하고 사라지고 '10'만 남게 된답니다.

[변수값 블록]
동그랗게 생긴 이 블록은 "지금 이 상자 안에 뭐가 들어있지?" 하고 안을 들여다보는 블록이에요. 지금 경험치가 몇 점인지 확인할 때 꼭 필요해요!

⚖️ 더하기 vs 정하기! 무엇이 다를까?

엔트리에서 변수를 다룰 때 가장 헷갈리는 두 가지 블록이 있어요. 시뮬레이터를 직접 눌러보며 차이를 알아볼까요?

💧 물_경험치 바구니 시뮬레이터

현재 물_경험치: 0
(바구니가 비어 있어요)
버튼을 눌러보세요!

💡 차이점 정리 노트:

  • ~만큼 더하기: 원래 상자에 들어있던 것에 추가로 더 담는 것이에요. (알을 클릭할 때마다 점수가 계속 늘어나요!)
  • ~로 정하기: 원래 상자에 뭐가 있었든 다 쏟아버리고, 딱 그 숫자만큼만 새롭게 담는 것이에요. (게임을 처음 시작할 때 0점으로 비우는 용도로 많이 써요!)

3. 실습: 나만의 성장 게임 코딩 🛠️

✅ 사전 준비

먼저 선생님이 준비한 기본 파일을 다운로드하고 엔트리에서 열어주세요!

💾 학생용 엔트리 파일 다운로드
  • 다운로드한 파일을 엔트리에서 [오프라인 작품 불러오기]로 열어주세요.
  • 엔트리 상단 메뉴의 [속성] -> [변수]에서 '물_경험치'를 만들어주세요.

📝 어떻게 코딩할까요?

💧 1. 물의 알 코딩하기

  1. 시작하기 버튼을 클릭했을 때: 게임이 시작되면 '물_경험치'를 0으로 정합니다. (상자 싹 비우기!)
  2. 오브젝트를 클릭했을 때: 마우스로 알을 누르면 '물_경험치'에 1만큼 더합니다. (경험치 쌓기!)
  3. 만일 '물_경험치'가 10이라면: 점수가 10점 꽉 찼는지 확인합니다. (이때 '변수값' 블록을 써요!)
  4. 다음 모양으로 바꾸기: 알을 깨고 드래곤이 나오게 합니다!
  5. '물_경험치'를 0으로 정하기: 다음 레벨(모양)을 위해 상자를 다시 비웁니다.

[이렇게 조립해 보세요!]

1. [시작] - 시작하기 버튼을 클릭했을 때
2. [자료] - 물_경험치를 (0)으로 정하기

3. [시작] - 오브젝트를 클릭했을 때
4. [자료] - 물_경험치에 (1)만큼 더하기
5. [흐름] - 만일 (물_경험치값 = 10) 이라면
6.     [생김새] - 다음 모양으로 바꾸기
7.     [자료] - 물_경험치를 (0)으로 정하기

🔥 2. 불의 알과 이끼의 알

물의 알과 똑같은 방법으로 코딩하되, 변수 이름만 '불_경험치', '이끼_경험치'로 바꿔주면 완성!

4. 정리 및 퀴즈 🏆

오늘 우리는 값을 저장하는 요술 상자인 변수를 사용해 게임을 만들었어요.

🤓 확인 퀴즈

Q1. 게임을 처음 시작할 때, 기존 점수를 싹 비우고 0점부터 시작하고 싶어요. 어떤 블록을 써야 할까요?

Q2. 변수 상자에 '사과'라는 글자가 들어있습니다. 여기에 '바나나'를 새롭게 넣으면 상자 안에는 무엇이 있을까요?

Q3. 게임 중 "지금 내 경험치가 몇 점이지?" 하고 변수 상자 안을 들여다보고 싶을 때 사용하는 동그란 블록의 이름은 무엇일까요?

Q4. '물_경험치' 상자에 현재 '5'가 들어있어요. 이때 [물_경험치에 (1)만큼 더하기] 블록을 실행하면 상자 안의 숫자는 어떻게 변할까요?

Q5. 게임을 만들 때 점수, 남은 시간, 내 이름 등을 기억하기 위해 사용하는 '이름표가 붙은 요술 상자'를 우리는 무엇이라고 부르나요?

✨ 나만의 도전 과제!

  • 진화할 때 멋진 효과음을 넣어보세요!
  • 경험치가 10이 아니라 20이 되어야 진화하도록 난이도를 높여보세요.
