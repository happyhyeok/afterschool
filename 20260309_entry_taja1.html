오늘의 핵심 개념

수많은 강릉의 명소 단어들을 하나하나 변수로 만들면 코드가 너무 길어지겠죠?
그래서 우리는 여러 개의 데이터를 한꺼번에 담는 ‘리스트’ 주머니를 사용합니다.

2단계: 엔트리 실습 도전!
미션 0. 필요한 오브젝트 추가하기
코딩 오브젝트 : 오브젝트 추가 - 배경, 단어(글상자), 안내자(엔트리봇 등)를 준비해요.
1. [오브젝트 추가하기]에서 원하는 배경을 선택하세요.
2. [글상자] 탭에서 내용을 비운 뒤 추가하세요.
3. 타자 입력창을 띄워줄 안내자 오브젝트를 하나 추가하세요.
미션 1. 리스트 및 변수 만들기
코딩 오브젝트 : 속성 탭 - 단어 리스트와 점수 변수를 생성해요.
1. [리스트 추가]: '강릉명소' 리스트를 만들고 아래 버튼을 눌러 단어를 복사해 넣으세요.
2. [변수 추가]: '점수' 변수를 추가하여 화면에 점수가 보이게 하세요.
미션 2. 무한 복제본 생성 알고리즘
코딩 오브젝트 : 글상자 - 계속해서 글상자의 복제본을 만들어요.
- 시작하기 버튼을 클릭했을 때
- 계속 반복하기 [ 2초 기다리기, 자신의 복제본 만들기 ]
미션 3. 복제본: 랜덤 단어 설정
코딩 오브젝트 : 글상자 - 복제된 글상자가 무작위 단어를 보여주게 해요.
- 복제본이 처음 생성되었을 때
- x좌표를 (-200 ~ 200) 사이의 무작위 위치로 이동
- 글상자 내용을 [강릉명소 리스트의 (1 ~ 리스트 항목 수) 사이의 무작위 수]번째 항목으로 바꾸기
미션 4. 복제본: 낙하 로직
코딩 오브젝트 : 글상자 - 단어가 위에서 아래로 떨어지게 만들어요.
- y좌표가 바닥(-150)보다 클 때까지 반복하기
- [ y좌표를 -3만큼 바꾸기 ]
- 반복이 끝나면 이 복제본 삭제하기
미션 5. 안내자: 입력 창 만들기
코딩 오브젝트 : 안내자 - 사용자가 타자를 칠 수 있는 창을 계속 띄워요.
- 시작하기 버튼을 클릭했을 때
- 계속 반복하기 [ "정답은?" 라고 묻고 대답 기다리기 ]
미션 6. 복제본: 정답 판별 및 점수
코딩 오브젝트 : 글상자 - 입력한 대답이 맞으면 점수를 올리고 삭제해요.
1. [속성] - [변수]에서 '점수' 변수를 먼저 생성하세요.
2. 미션 4의 낙하 반복문 안에 다음을 추가하세요:
- 만일 [ 대답 = 자신의 글상자 내용 ] 이라면
- [ '점수'에 10만큼 더하기 ]
- [ 이 복제본 삭제하기 ]
