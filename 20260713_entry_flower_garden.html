마우스와 키보드로 꽃밭 가꾸기

마우스로 꽃을 심고 키보드로 꽃의 크기와 종류를 바꾸어 봅니다.

완성 작품 미리보기

마우스와 키보드를 이용해 여러 가지 꽃으로 꽃밭을 꾸며 봅니다.

수업 목표

✅ 마우스를 따라 움직이는 꽃 만들기
✅ 클릭한 위치에 꽃 심기
✅ 방향키로 꽃 크기 바꾸기
✅ 스페이스바로 꽃 종류 바꾸기

오늘의 미션

풀 배경 위에 크기와 종류가 다른 꽃을 10개 이상 심어 나만의 꽃밭을 완성합니다.

핵심 원리: 마우스 클릭과 키보드 입력은 프로그램에 명령을 전달하는 입력입니다.

1풀 배경과 꽃 준비하기

지난 시간에는 꽃잎을 반복해서 찍고 회전하여 꽃송이를 만들었습니다. 이번 시간에는 마우스와 키보드를 이용해 원하는 곳에 여러 가지 꽃을 심어 봅니다.

  1. 기존 엔트리봇 오브젝트를 삭제합니다.
  2. 오브젝트 추가하기를 클릭합니다.
  3. 라이브러리 선택 → 배경 → 자연에서 풀 배경을 추가합니다.
  4. 라이브러리 선택 → 식물 → 꽃에서 데이지 오브젝트를 추가합니다.
주의: 실행 중에는 작품을 수정할 수 없습니다. 먼저 정지하기를 누른 후 수정합니다.

2꽃이 마우스를 따라 움직이게 하기

꽃이 마우스 포인터를 계속 따라 움직이도록 블록을 연결합니다.

만드는 방법: 엔트리봇 위치로 이동하기 블록의 드롭다운을 클릭한 뒤 마우스 포인터를 선택하면 마우스 포인터 위치로 이동하기가 됩니다.

▶ 시작하기 버튼을 클릭했을 때 └ 계속 반복하기    └ 마우스 포인터 위치로 이동하기

확인 질문: 마우스를 움직이면 꽃은 어떻게 움직이나요?

정답: 꽃이 마우스 포인터를 따라 움직입니다.

3클릭한 위치에 꽃 심기

마우스를 클릭할 때마다 현재 위치에 꽃이 남도록 만듭니다.

▶ 마우스를 클릭했을 때 └ 도장찍기

실습 미션: 장면의 서로 다른 위치에 꽃을 5개 심어 봅니다.

4방향키로 꽃 크기 바꾸기

위쪽 방향키를 누르면 꽃이 커지고, 아래쪽 방향키를 누르면 꽃이 작아지도록 만듭니다.

블록 1

만드는 방법: q 키를 눌렀을 때 블록의 드롭다운을 클릭한 뒤 위쪽 방향키를 선택합니다.

블록 2

▶ 위쪽 방향키를 눌렀을 때 └ 크기를 10만큼 바꾸기 ▶ 아래쪽 방향키를 눌렀을 때 └ 크기를 -10만큼 바꾸기

꽃을 심기 전에 방향키를 눌러 원하는 크기로 바꿉니다.

실습 미션: 큰 꽃, 중간 꽃, 작은 꽃을 각각 한 개 이상 심습니다.

5스페이스바로 꽃 종류 바꾸기

꽃 오브젝트에 백일홍과 해바라기 모양을 추가합니다.

  1. 모양 탭에서 모양 추가를 클릭합니다.
  2. 식물 → 꽃에서 백일홍과 해바라기를 추가합니다.
  3. 명령 블록을 연결합니다.
▶ 스페이스 키를 눌렀을 때 └ 다음 모양으로 바꾸기

꽃의 변화: 데이지 → 백일홍 → 해바라기 → 데이지

실습 미션: 세 종류의 꽃을 모두 한 번 이상 심습니다.

연습 문제 1

꽃의 크기를 더 빠르게 바꾸기

문제: '크기를 10만큼 바꾸기'의 값을 20으로 바꾸어 실행해 봅니다.

생각 질문: 값을 크게 바꾸면 꽃의 크기는 어떻게 달라질까요?

정답: 키를 한 번 눌렀을 때 크기가 더 많이 변합니다.

연습 문제 2

다른 오브젝트로 꾸미기

꽃 대신 나비, 별, 과일 등 원하는 오브젝트를 추가해 봅니다.

새 오브젝트에 적용할 기능내용
마우스 따라가기마우스 포인터 위치로 이동합니다.
클릭하여 도장찍기클릭한 위치에 오브젝트를 남깁니다.
방향키로 크기 변경키보드 입력으로 크기를 바꿉니다.
스페이스바로 모양 변경다음 모양으로 바꿉니다.

확인 질문

도전! 나만의 꽃밭 꾸미기

조건을 모두 만족하는 나만의 꽃밭을 만들어 봅니다.

조건확인
꽃을 10개 이상 심기장면에 꽃이 충분히 있는지 확인합니다.
꽃 종류 3가지 이상 사용하기데이지, 백일홍, 해바라기를 모두 사용합니다.
크기가 다른 꽃 넣기큰 꽃, 중간 꽃, 작은 꽃을 섞습니다.
빈 곳과 가득 찬 곳을 알맞게 배치하기꽃밭이 보기 좋게 균형을 맞춥니다.

추가 도전

꽃이 아닌 오브젝트를 한 종류 더 넣어 꽃밭에 이야기를 만들어 봅니다.

예: 나비가 찾아온 꽃밭, 별이 피어나는 밤의 꽃밭, 과일이 열리는 신기한 정원

추가 미션

꽃밭을 완성했나요?
아래에서 마음에 드는 미션을 골라 더 멋진 작품으로 발전시켜 보세요.

기본 활동을 완료한 학생만 선택해서 도전합니다. 세 미션 중 하나만 성공해도 훌륭합니다. 더 해 보고 싶다면 다음 미션에 도전해 보세요.

🧹 지우기 버튼 만들기

키보드의 R 키를 누르면 지금까지 심은 꽃이 모두 사라지게 만들어 보세요.

▶ R 키를 눌렀을 때 └ 도장 모두 지우기
💡 막혔을 때 먼저 확인해요

'도장 모두 지우기' 블록은 [붓] 블록 꾸러미에서 찾아보세요. R 키가 보이지 않으면 키 선택 부분을 눌러 R을 선택하세요.

  1. 꽃을 여러 개 심습니다.
  2. R 키를 누릅니다.
  3. 심어 놓은 꽃이 모두 사라지면 성공입니다.

실수해도 괜찮아요. R 키를 누르면 꽃밭을 깨끗하게 다시 시작할 수 있어요.

🌻 기울어진 꽃 만들기

왼쪽 방향키와 오른쪽 방향키를 눌러 꽃의 방향을 바꾼 뒤 심어 보세요.

▶ 왼쪽 방향키를 눌렀을 때 └ 방향을 -15만큼 회전하기 ▶ 오른쪽 방향키를 눌렀을 때 └ 방향을 15만큼 회전하기
💡 막혔을 때 먼저 확인해요

꽃이 옆으로 움직인다면 '이동하기'가 아니라 '회전하기' 블록을 사용했는지 확인하세요. 숫자 15를 더 크게 하면 한 번에 더 많이 기울어집니다.

  1. 왼쪽 또는 오른쪽 방향키를 누릅니다.
  2. 꽃이 기울어진 것을 확인합니다.
  3. 마우스를 클릭하여 기울어진 꽃을 심습니다.
  4. 서로 다른 방향의 꽃을 3개 이상 심으면 성공입니다.

같은 꽃도 방향을 다르게 하면 바람에 흔들리는 꽃밭처럼 보일 수 있어요.

🎲 크기가 제각각인 꽃 심기

마우스를 클릭할 때마다 꽃의 크기가 무작위로 달라진 뒤 그 위치에 꽃이 심어지게 만들어 보세요.

기존 코드 ▶ 마우스를 클릭했을 때 └ 도장찍기 수정할 코드 ▶ 마우스를 클릭했을 때 ├ 크기를 50부터 150 사이의 무작위 수로 정하기 └ 도장찍기
💡 막혔을 때 먼저 확인해요

무작위 수 블록은 [계산] 블록 꾸러미에서 찾을 수 있어요. 무작위 수 블록을 '크기를 정하기' 블록의 숫자 칸 안에 끼워 넣으세요. 반드시 크기를 먼저 바꾼 다음 도장을 찍어야 합니다.

  1. 같은 크기의 꽃을 여러 번 심어 봅니다.
  2. 수정한 코드를 실행합니다.
  3. 클릭할 때마다 꽃의 크기가 다르게 나타나면 성공입니다.

코드의 순서가 중요해요. 도장을 먼저 찍으면 바뀐 크기는 다음 꽃부터 적용됩니다.

선생님께 질문하기 전에 확인해요
