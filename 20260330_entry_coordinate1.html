🌊 3차시: 좌표 기초 - 바닷가 친구들 움직이기

1. 도입: 마법의 지도, 좌표! 🗺️

안녕, 친구들! 오늘은 바닷가에 놀러 온 꽃게와 고등어를 엔트리에서 움직여 볼 거예요.

컴퓨터 화면은 엄청 넓은 운동장과 같아요. 꽃게에게 "오른쪽으로 가!"라고 말하면, 컴퓨터는 "얼마나 가야 해? 어디로 가야 해?"라고 헷갈려 한답니다. 이때 정확한 위치를 알려주는 마법의 주소가 바로 좌표(X, Y)랍니다!

2. 전개: 움직이기 vs 위치로 이동하기 🤔

좌표에는 두 가지 중요한 규칙이 있어요.

  • X 좌표: 가로 방향(왼쪽, 오른쪽)이에요. ↔️
  • Y 좌표: 세로 방향(위, 아래)이에요. ↕️

코딩을 하다 보면 "~만큼 움직이기"와 "~위치로 이동하기" 두 블록을 보게 될 거예요. 이 둘은 어떻게 다를까요? 아래 자동차🚗 버튼을 직접 눌러보며 확인해 봐요!

🎮 직접 눌러보며 차이점 알아보기!

현재 자동차의 X 좌표: 0

X: -100 (왼쪽) X: 0 (가운데) X: 100 (오른쪽)
🚗

버튼을 눌러보세요! 어떤 차이가 있나요?

💡 비밀 노트:
- ~만큼 움직이기: 지금 있는 곳에서부터 덧셈(+)을 해서 앞으로 나아가는 거예요! (누를 때마다 계속 감)
- ~위치로 이동하기: 지금 어디에 있든 상관없이, 내가 정해준 그 자리로 '순간이동' 하는 거예요! (여러 번 눌러도 그 자리에 멈춰 있음)

3. 실습: 바닷가 친구들 코딩하기 💻

✅ 사전 준비 (먼저 해주세요!)

  • 네이버 엔트리(playentry.org)에 접속하고 로그인해 주세요.
  • 상단 메뉴에서 [만들기] - [작품 만들기]를 클릭해요.
  • 화면 가운데 있는 귀여운 엔트리봇은 'X' 버튼을 눌러 지워주세요.

✅ 필요한 오브젝트 추가하기

오브젝트 추가하기(+) 버튼을 누르고 아래 4가지를 검색해서 추가해 주세요!

  • 🐟 물고기
  • 🦀 꽃게
  • ☁️ 구름
  • 🏖️ 해변가 (배경)

📝 어떻게 코딩할까요?

아래 의사코드의 순서에 따라 블록을 차례대로 연결해 보세요.

☁️ 1. 구름 코딩하기

  1. 시작하기 버튼을 클릭하면
  2. 크기를 70으로 정해요.
  3. x: 210, y: 95 위치로 이동해요. (오른쪽 위 하늘로 가요!)
  4. 모양을 보여요.
  5. 15초 동안 x: -210, y: 95 위치로 이동해요. (왼쪽으로 천천히 흘러가요)
  6. 모양을 숨겨요.

🐟 2. 고등어 코딩하기

  1. 시작하기 버튼을 클릭하면
  2. 크기를 45로 정해요.
  3. x: -160, y: 30 위치로 이동해요.
  4. 5초 기다려요.
  5. 1초 동안 구름 위치로 이동해요. (구름한테 날아가요!)
  6. 1초 동안 x: 160, y: 30 위치로 이동해요.

🦀 3. 꽃게 코딩하기 (키보드로 조종해요!)

📌 꽃게가 특정 위치로 이동해요.

  1. 시작하기 버튼을 클릭하면
  2. 크기를 50으로 정해요.
  3. x: -140, y: -65 위치로 이동해요.

📌 키보드 화살표키를 누르면, 위, 아래, 왼쪽, 오른쪽으로 요리조리 움직여요.

  1. 위쪽 화살표 키를 누르면 y 좌표를 2만큼 바꿔요.
  2. 아래쪽 화살표 키를 누르면 y 좌표를 -2만큼 바꿔요.
  3. 오른쪽 화살표 키를 누르면 x 좌표를 2만큼 바꿔요.
  4. 왼쪽 화살표 키를 누르면 x 좌표를 -2만큼 바꿔요.

🌟 선생님의 꿀팁: 꽃게가 움직이는 숫자가 '2'라서 너무 느리게 느껴진다면, 숫자를 '10'이나 '-10'으로 바꿔보세요! 슝슝 빠르게 움직일 거예요.

4. 정리 및 퀴즈 🏆

오늘 우리는 화면 안에서 물건의 위치를 알려주는 'X 좌표(가로)'와 'Y 좌표(세로)'를 배웠어요.

🤓 확인 퀴즈 (정답을 클릭해 보세요!)

Q1. 꽃게가 위, 아래로 움직이려면 어떤 좌표를 바꾸어 주어야 할까요?

Q2. 꽃게가 오른쪽으로 이동하려면 X 좌표는 어떻게 되어야 할까요?

Q3. 지금 있는 곳에서 계속 앞으로 나아가려면 어떤 블록을 써야 할까요?

Q4. 화면 정중앙으로 순간이동하려면 X, Y 좌표를 각각 몇으로 해야 할까요?

Q5. 고등어가 5초 동안 제자리에서 기다리게 하려면 어떤 블록이 필요할까요?
