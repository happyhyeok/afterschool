1 핵심 알고리즘 설명
📍 좌표와 이동
• x좌표 : 가로 좌표. 오른쪽 갈수록 커지고, 왼쪽으로 갈수록 작아져요.
• y좌표 : 세로 좌표. 위로 갈수록 커지고, 아래로 갈수록 작아져요.
• ~ 위치로 이동하기 : 지정된 위치로 순간이동해요.
• ~ 만큼 바꾸기 : 현재 위치에서 지정한 숫자만큼 좌표를 더하거나 빼서 움직임을 만들어요.
⚡ 복제본과 신호
• 복제본 : 하나의 오브젝트로 여러 개가 나타나는 효과를 만들고 싶을 때 사용해요.
• 신호 : 오브젝트끼리 서로 소식을 주고받을 때 사용해요. (예: 게임 끝!, 2단계 시작!)
📦 변수(Variable)란?
변수는 데이터를 담는 '이름표가 붙은 상자'와 같아요.
게임 중 언제든지 변할 수 있는 값(점수, 남은 시간 등)을 상자에 저장해두고 필요할 때 꺼내 쓰거나 내용을 바꿀 수 있습니다.

2 작품 구성 및 오브젝트 목록
  • 🚀

    우주선 (주인공)

    키보드 화살표로 조종하며 보석을 먹고 운석을 피합니다.

  • 💎

    보석 (아이템)

    하늘에서 떨어지며, 우주선에 닿으면 점수가 올라갑니다.

  • 🌑

    운석 (장애물)

    하늘에서 떨어지며, 우주선에 닿으면 게임이 종료됩니다.

  • 🌌

    배경 (우주)

    게임의 무대이며, 점수에 따라 모양이 바뀝니다.

  • 💬

    글상자 (알림)

    게임 시작, 스테이지 안내, 결과 메시지를 표시합니다.

3 [글상자] 알고리즘(1) 게임 시작 알리기

게임이 시작될 때 사용자에게 안내 메시지를 보여줍니다.

[글상자 오브젝트] ① 시작하기 버튼을 클릭했을 때 ② 글상자 내용을 [3초 뒤에 게임이 시작됩니다!]로 바꾸기 ③ 3초 기다리기 ④ 글상자 내용을 [보석을 모으세요!]로 바꾸기 ⑤ 1초 기다리기 후 모양 숨기기
4 [우주선] 조종 알고리즘

좌우 화살표 키를 이용해 우주선을 움직여요.

[우주선 오브젝트] ① 시작하기 버튼을 클릭했을 때 ② 계속 반복해서 └ 만일 왼쪽 화살표 키를 눌렀다면 -> x 좌표를 -5만큼 바꾸기 └ 만일 오른쪽 화살표 키를 눌렀다면 -> x 좌표를 5만큼 바꾸기
5 [보석] 복제 및 점수 획득

5-1. 보석이 하늘에서 떨어져요.

[보석 오브젝트] ① 시작하기 버튼 클릭 시: 숨기기 ② 무한 반복: 2초 기다리기 후 자신의 복제본 만들기 ③ 복제본 생성 시: x [무작위], y [150] 위치 이동 후 보이기 ④ 무한 반복: y 좌표를 -3만큼 바꾸기

5-2. 우주선이 보석을 먹으면 점수가 올라가요.

[보석 복제본 코드 안] ① 만일 우주선에 닿았는가? └ 점수 변수에 1만큼 더하기 └ 이 복제본 삭제하기
6 [운석] 장애물 알고리즘

운석이 하늘에서 떨어지며 우주선과 충돌하면 게임이 끝납니다.

[운석 오브젝트] ① 보석과 유사하게 복제본 생성 (속도는 더 빠르게!) ② 만일 우주선에 닿았는가? └ 신호 [게임 종료] 보내기
