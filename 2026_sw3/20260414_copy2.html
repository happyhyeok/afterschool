미션

진짜 바탕화면 정리 미션 📂

연습을 잘 마쳤나요? 이제 진짜 컴퓨터 바탕화면에서 실력을 보여주세요!
바탕화면에 길을 잃은 동물 친구들을 단축키로 안전한 '동물농장' 폴더로 이사시켜 주세요!

1 바탕화면에 새 폴더 만들기

바탕화면 빈 곳을 마우스 오른쪽 클릭 → '새로 만들기' → '폴더'를 누르고 이름을 동물농장으로 바꿔요.

2 동물 사진 잘라내기 (순간이동 준비)

바탕화면에 있는 '사자', '호랑이', '토끼' 사진을 드래그해서 모두 선택한 후, Ctrl + X 를 눌러요. 사진이 약간 투명해질 거예요!

3 동물농장으로 이사시키기

'동물농장' 폴더를 더블클릭해서 연 다음, 빈 곳을 한 번 클릭하고 Ctrl + V 를 눌러요! 동물들이 뿅 나타나면 성공!
