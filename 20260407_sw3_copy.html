1 마법이 필요한 순간! 🤔

상상해 보세요! 생일 파티 초대장 쓰기 🎂

내 생일 파티에 친구 10명을 초대하려고 해요.
도화지에 "내 생일 파티에 놀러와! 토요일 오후 2시야"라는 똑같은 편지를 10번이나 직접 손으로 쓰려면 팔이 너무 아프겠죠?

밤하늘에 별 100개 그리기 ⭐

도화지에 정말 예쁜 별을 하나 그렸어요.
이 별을 밤하늘 가득 100개로 채우고 싶어요.
하나하나 다시 똑같이 그리려면 시간도 오래 걸리고 모양도 삐뚤빼뚤해질 수 있어요.

이럴 때 컴퓨터 세상에서는 단 한 번만 만들고,
'복사 마법'을 써서 1초 만에 똑같은 쌍둥이를 무한대로 만들어 낼 수 있답니다!

2 마우스 우클릭 마법 삼총사와 단축키 🪄

파일 위에서 마우스 오른쪽 버튼을 누르면 마법의 메뉴가 나타나요. 키보드를 사용하면 마법을 더 빨리 쓸 수도 있어요!

복사하기 파일은 원래 자리에 두고
똑같은 쌍둥이를 복제해요!
마법의 키보드 (다음 시간에 배울 내용) Ctrl + C
잘라내기 원래 자리에서 지우고
새로운 곳으로 이사 가요!
마법의 키보드 (다음 시간에 배울 내용) Ctrl + X
붙여넣기 복사하거나 잘라낸 파일을
원하는 곳에 짠! 나타나게 해요.
마법의 키보드 (다음 시간에 배울 내용) Ctrl + V
마법 종류 원래 있던 파일은? 새로운 곳에 파일은?
복사 그대로 남아있어요 🙆‍♂️ 쌍둥이가 생겨요! ✌️
잘라내기 싹 사라져요 🙅‍♀️ 이사 와서 나타나요! 🚚

🤫 컴퓨터의 비밀 주머니, '클립보드(Clipboard)'

복사하거나 잘라낸 파일은 어디로 갈까요? 바로 다른 곳에 짠! 하고 나타나지 않죠?
컴퓨터는 우리가 '붙여넣기'를 할 때까지 파일을 잃어버리지 않게 기억하려고 '클립보드'라는 보이지 않는 투명 주머니에 잠시 보관해 둔답니다!
