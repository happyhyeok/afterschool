🎲 모양 맞추기 게임 만들기

무작위 모양 맞추기 게임

버튼을 누르면 물건 모양이 빠르게 바뀝니다. 다시 누르면 모양이 멈추고, 세 물건의 모양이 모두 같은지 확인합니다.

🎯
수업 목표 모양을 아무렇게나 바꾸고, 버튼으로 시작과 멈춤을 조절하며, 세 모양이 같은지 판단하는 게임을 만들 수 있습니다.
1. 핵심 생각 2. 작품 모습 3. 준비하기 4. 버튼 코딩 5. 물건 코딩 6. 복제하기 7. 도전 미션
1

🧠 오늘 배울 핵심 생각

이번 작품은 “모양 바꾸기 → 버튼으로 멈추기 → 같은 모양인지 확인하기” 순서로 만듭니다.

아무 숫자 뽑기

컴퓨터가 1, 2, 3 중에서 하나를 골라 모양을 바꿉니다. 그래서 누를 때마다 결과가 달라질 수 있습니다.

신호 보내기

버튼이 물건에게 “바꿔라”, “멈춰라”처럼 짧은 말을 보내 움직임을 알려 줍니다.

만약 판단하기

상태가 0이면 시작하고, 상태가 1이면 멈추게 합니다. 상황에 따라 다른 행동을 하게 만드는 방법입니다.

조건 묶기

세 물건이 모두 같은지 보려면 조건을 여러 개 연결해서 함께 확인해야 합니다.

💡
왜 “그리고”가 필요할까요?

세 모양이 모두 같아야 정답입니다. 그래서 첫 번째와 두 번째가 같고, 두 번째와 세 번째도 같은지 함께 확인합니다.

연결 말 뜻 언제 맞다고 볼까? 게임에서 쓰는 예
그리고 두 조건을 모두 만족해야 합니다. 두 가지가 다 맞을 때만 실행합니다. 물건1 = 물건2 그리고 물건2 = 물건3
세 물건이 모두 같아야 정답입니다.
또는 조건 중 하나만 만족해도 됩니다. 두 가지 중 하나라도 맞으면 실행합니다. 비가 온다 또는 눈이 온다
둘 중 하나라도 맞으면 우산을 챙깁니다.

🎮 작품 모습

⭐
🍀
💎

수업에서 만들 게임의 모습을 간단히 나타낸 그림입니다.

게임 방법

버튼을 한 번 누르면 물건 모양이 빠르게 바뀝니다. 다시 누르면 모양이 멈춥니다.

성공 조건

세 물건의 모양이 모두 같으면 성공입니다. 하나라도 다르면 다시 도전합니다.

필요한 데이터

변수: 상태 (0 = 멈춤 / 1 = 움직이는 중)
신호: 바꿔라, 멈춰라

🛠️ 준비하기

🔗
학생용 작품 복제하기

아래 링크를 열고 작품을 복제한 뒤, 필요한 오브젝트와 데이터를 준비합니다.

학생용 작품 바로가기
  1. 오브젝트 준비
    엔트리봇을 삭제하고, 물건1과 버튼 오브젝트를 준비합니다.
  2. 물건1 모양 준비
    물건1에 서로 다른 모양을 3개 넣습니다.
  3. 변수 만들기
    변수 상태를 만듭니다. 처음 값은 0으로 둡니다.
  4. 신호 만들기
    신호 바꿔라, 멈춰라를 만듭니다.

🔴 버튼 코딩

버튼은 게임을 시작하고 멈추게 합니다. 모양이 멈춘 뒤에는 세 모양이 같은지도 확인합니다.

① 시작과 멈춤 정하기

현재 상태 값 버튼을 눌렀을 때 보낼 신호 새 상태 값 뜻
0 바꿔라 1 멈춰 있으니 모양을 바꾸기 시작합니다.
1 멈춰라 0 모양이 바뀌는 중이니 멈춥니다.

버튼을 클릭했을 때

만일 상태 = 0 이라면
바꿔라 신호 보내기
상태를 1로 정하기
아니면
멈춰라 신호 보내기
상태를 0으로 정하기

② 정답 확인하기

멈춰라 신호를 받았을 때

만일 물건1 모양 = 물건2 모양 그리고 물건2 모양 = 물건3 모양 이라면
“맞췄습니다!” 말하기
아니면
“틀렸습니다!” 말하기
생각해 보기 · 세 모양이 모두 같으려면 물건1과 물건2가 같고, 물건2와 물건3도 같아야 합니다. 그래서 두 조건을 그리고로 연결합니다.

🪄 마무리: 오브젝트 복제하기

코딩이 끝난 물건을 복사해서 3개로 만듭니다.

🎩 물건1 복제 🎩 물건2 복제 🎩 물건3
  1. 물건1을 우클릭하고 복제를 2번 눌러 물건2, 물건3을 만듭니다.
  2. 실행 화면에서 물건 3개가 겹치지 않게 가로로 나란히 놓습니다.
  3. 시작하기 버튼을 눌러 모양이 바뀌고 멈추는지 확인합니다.

🚀 도전 미션

🎯 미션 1: 효과음 넣기

목표: 맞췄을 때와 틀렸을 때 다른 소리가 나게 만듭니다.

힌트 · [소리] 탭에서 소리를 추가한 뒤, 정답을 확인하는 블록 안에 소리 재생하기 블록을 넣습니다.

🎯 미션 2: 점수판 만들기

힌트 · 점수 변수를 새로 만들고, 정답과 오답을 확인하는 부분에 점수 값 ~만큼 더하기 블록을 넣습니다.
