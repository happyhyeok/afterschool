4. 로봇어(의사코드) 조립 설명서

아래의 상세 코드 보기 버튼을 눌러 각 친구들의 코드를 조립해 보세요!

우주선 조립하기

① 화살표 키로 움직이고, 스페이스바로 무기를 발사해요.

[시작하기 버튼을 클릭했을 때] 계속 반복하기 ➔ 만일 [위쪽 화살표] 키가 눌러져 있는가? 라면 Y좌표를 10만큼 바꾸기 ➔ // 아래, 왼쪽, 오른쪽 화살표도 똑같이 만들어요! (왼쪽/아래는 -10)
[시작하기 버튼을 클릭했을 때] 계속 반복하기 ➔ 만일 [스페이스] 키가 눌러져 있는가? 라면 [무기] 자신의 복제본 만들기

무기(총알) 조립하기

① 생성되면 오른쪽으로 날아가고, 모기를 맞추면 10점을 얻어요.

[시작하기 버튼을 클릭했을 때] 계속 반복하기 ➔ [우주선] 위치로 이동하기
[복제본이 처음 생성되었을 때] [오른쪽 벽]에 닿았는가? 가 될 때까지 반복하기 ➔ X좌표를 15만큼 바꾸기 ➔ 만일 [적전투기]에 닿았는가? 라면 ➔ [점수]에 10만큼 더하기 ➔ 이 복제본 삭제하기 ➔ 이 복제본 삭제하기 // 벽에 닿으면 삭제

적전투기 조립하기

① 계속 나타나서 왼쪽으로 날아오고, 우주선과 닿으면 충돌 신호를 보내요.

[시작하기 버튼을 클릭했을 때] ➔ 모양 숨기기 계속 반복하기 ➔ 1.5 ~ 3.5초 무작위 수 초 기다리기 ➔ 자신의 복제본 만들기
[복제본이 처음 생성되었을 때] // 날아오기 조립 ➔ 모양 보이기 ➔ X:170, Y:(무작위 수) 위치로 이동하기 [왼쪽 벽]에 닿았는가 될 때까지 반복하기 ➔ X좌표를 -5만큼 바꾸기 ➔ 만일 [우주선]에 닿았는가? 라면 ➔ [충돌] 신호 보내기
[복제본이 처음 생성되었을 때] // 총알에 맞았을 때 계속 반복하기 ➔ 만일 [무기]에 닿았는가? 라면 ➔ 색깔 효과 주기, 0.1초 기다리기, 이 복제본 삭제하기

폭발 효과 조립하기

① 평소엔 숨어있다가, 충돌 신호를 받으면 쾅! 하고 모든 게임을 멈춰요.
