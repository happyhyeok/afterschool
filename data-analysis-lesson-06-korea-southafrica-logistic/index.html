팀 종합점수란?

실제 축구 분석에서는 경기력을 한 가지 숫자로만 판단하지 않습니다.
슈팅, 유효슈팅, 득점 기회, 실점, 최근 경기 흐름 같은 여러 데이터를 함께 봅니다.

오늘은 쉽게 실험할 수 있도록 이 여러 정보를 합쳐 “팀 종합점수”라고 부르겠습니다.

팀 종합점수 안내
90 아주 좋은 흐름
80 좋은 흐름
70 비슷한 흐름
60 조금 어려운 흐름

3

로지스틱 회귀는 어떻게 판단할까요?

1. AI는 정답이 붙은 예시 데이터를 먼저 봐요

AI는 처음부터 답을 아는 것이 아닙니다. 먼저 정답이 붙어 있는 예시 데이터를 보고 배웁니다.

정답이 붙은 예시 데이터
한국팀 종합점수 남아공팀 종합점수 정답
85 65 대한민국 승리 가능, 1
75 75 접전 예상, 0
65 85 접전 예상, 0

여기서 1은 대한민국 승리 가능, 0은 접전 예상이라는 뜻입니다.
AI가 보는 데이터는 그냥 숫자표가 아니라 정답이 붙어 있는 연습 문제입니다.

2. AI는 데이터를 점처럼 생각해요

AI는 예시 데이터를 그래프 위의 점처럼 생각합니다.

  • 한국팀 85, 남아공팀 65이 점은 1
  • 한국팀 75, 남아공팀 75이 점은 0
  • 한국팀 65, 남아공팀 85이 점은 0

이렇게 점마다 정답이 붙어 있습니다. AI는 “어떤 점들은 1이구나”, “어떤 점들은 0이구나”를 먼저 배웁니다.

3. AI는 가장 잘 나누는 기준을 찾아요

예시 데이터를 가장 잘 나누는 기준 찾기

로지스틱 회귀는 1과 0을 가장 잘 나누는 기준을 찾습니다. 이 기준은 사람이 직접 정하는 것이 아닙니다. AI가 여러 예시 데이터를 비교하면서 틀린 판단이 적어지는 방향으로 기준을 찾아갑니다.

기준선은 AI가 예시 데이터를 보고 찾아낸 분류 기준입니다.

4. AI는 가능성을 생각해요

새로운 점수가 들어오면 AI는 그 점수가 1에 가까운지, 0에 가까운지 생각합니다. 축구에서 흐름이 좋으면 승리 가능성이 높아질 수 있지만, 가능성은 100%를 넘을 수 없고 0%보다 작아질 수도 없습니다.

AI의 가능성 판단
AI의 생각 결과
1에 가까움 대한민국 승리 가능
0에 가까움 접전 예상

5. 마지막에는 1 또는 0으로 판단해요

상황별 AI 판단
상황 AI 판단
한국팀 종합점수가 높고 남아공팀 종합점수가 낮음 1에 가까움
두 팀 점수가 비슷함 0에 가까움
남아공팀 종합점수가 더 높음 0에 가까움
핵심 정리 로지스틱 회귀는 정답이 붙은 예시 데이터를 보고 1과 0을 나누는 기준을 찾습니다. 그리고 새로운 점수가 들어오면 어느 쪽에 가까운지 보고 결과를 판단합니다.

이 결과는 실제 경기 결과를 맞히는 예언이 아닙니다. 수업용 데이터를 보고 AI가 분류하는 방법을 배우는 활동입니다.
