🎯 우리의 임무

작년에 배운 '코딩 마법 3가지(순차, 반복, 조건)'를 활용하여, 엔트리봇이 무사히 미로를 탈출할 수 있는 코드를 완성하세요!

💡

💡 3단계: 스스로 탐구하기 (문제 작게 나누기)

한 번에 미로를 끝까지 탈출하려고 하면 너무 어려워요! '큰 문제'를 해결하기 쉬운 '작은 문제 7가지'로 나누어서 순서대로 하나씩 해결해 봅시다. (일찍 끝난 친구들을 위한 보너스 미션도 숨어있어요!)

🤔 어떻게 움직일지 방법 정하기

1. 주인공을 어떻게 움직이게 만들까요?

움직이는 방법에 맞추어 명령해 주어야 해요. 왼쪽의 움직임 설명과 오른쪽의 알맞은 명령 블록을 하나씩 클릭해서 짝지워 보세요!

👉 이번 미션에서는 '키보드 화살표키'로 움직이게 만들 거예요!

💻 방향키 코딩하기

주인공을 움직이는 블록은 크게 2가지 방법이 있어요. 어떻게 다른지 확인해 볼까요?

🧭 방법 1. '이동방향'으로 움직이기

주인공이 바라보는 방향(화살표)을 먼저 바꾼 다음, 그 방향으로 나아가는 방법이에요.

예) "오른쪽을 봐!" ➔ "10만큼 가!"

🗺️ 방법 2. '좌표'로 움직이기 ⭐ 이번 미션 추천!

화면의 숨겨진 눈금자(좌표)를 이용해 바라보는 방향을 바꾸지 않고 그대로 옆으로, 위아래로 움직이는 방법이에요.

💡 가로(왼쪽, 오른쪽)로 움직일 때는 X 좌표,
💡 세로(위, 아래)로 움직일 때는 Y 좌표를 사용해요.

3. 오른쪽으로 가기

🎯 미션: 위에서 배운 '좌표 바꾸기' 블록을 사용해서 오른쪽 화살표 키를 눌렀을 때 주인공이 오른쪽으로 이동하게 코딩해 보세요.

4. 나머지 방향 완성하기 (X, Y 좌표 생각하기)

오른쪽으로 움직이려면 X 좌표를 커지게(+) 해야 해요. 그럼 다른 방향은 어떻게 해야 할까요?

➡ 오른쪽
X 좌표를 + 만큼
⬅ 왼쪽
X 좌표를 - 만큼
⬆ 위쪽
Y 좌표를 + 만큼
⬇ 아래쪽
Y 좌표를 - 만큼

🎯 미션: 위 힌트를 보고 왼쪽, 위쪽, 아래쪽 화살표 키를 눌렀을 때의 코드도 스스로 완성해 보세요!

▶ 도움이 필요하다면! (상세 가이드 보기)

1. [시작] 탭에서 q 키를 눌렀을 때 블록을 가져와요.

2. 블록의 'q' 글자를 클릭해서 '오른쪽 화살표' 등 원하는 방향으로 바꿔주세요.

3. [움직임] 탭에서 x 좌표를 10 만큼 바꾸기(또는 y 좌표) 블록을 가져와서 아래에 끼워 넣어요.

4. 같은 방법으로 왼쪽(x 좌표를 -10), 위쪽(y 좌표를 10), 아래쪽(y 좌표를 -10) 블록도 만들어 보세요!

🧱 미로 벽 통과 막기

5. 미로 벽 통과를 막는 방법 정하기

미로 벽 통과를 막는 방법은 여러가지가 있지만, 크게 2가지 방법을 생각할 수 있어요.

1) 😭 처음 시작 위치로 돌아가게 만든다.

2) 🛑 벽을 뚫지 못하게 제자리에 멈추게 만든다.

👉 우리는 이 중에서 '벽을 뚫지 못하게' 만들 거예요!

벽을 뚫지 못하게 하려면!!

💡 '만약 ~라면' 조건 마법을 사용해서, 벽에 닿았을 때 반대 방향으로 아주 살짝 튕겨나가게 만들면 벽을 뚫을 수 없어요!

🎯 미션: 화살표 키를 눌러도 주인공이 벽을 뚫고 나가지 못하게 코딩해 보세요.

▶ 도움이 필요하다면! (상세 가이드 보기)

1. [흐름] 탭에서 만약 참(이)라면 블록을 가져와서 방향키 코드 바로 아래에 붙여요.

2. [판단] 탭에서 마우스 포인터에 닿았는가? 블록을 가져와 '참' 자리에 쏙 넣어요.

3. '마우스 포인터' 글자를 클릭해서 '미로(벽)'으로 바꿔주세요.

4. 벽에 닿았을 때 튕기게 하려면, 이동했던 방향과 반대 방향의 좌표 블록을 만약 블록 안에 넣어요.
(예: 오른쪽 이동 코드 안에는 x 좌표를 -10 만큼 바꾸기를 넣어요!)

🏁 목적지에 닿았을 때

6. 목적지에 닿으면 어떻게 할까요?

탈출에 성공했을 때 일어날 재미있는 일들을 상상해서 코딩해 보세요!

✨ 성공 이벤트 예시:
  • "만세! 탈출 성공!"이라고 2초 동안 말하기
  • 신나는 박수 소리 재생하기
  • 주인공 오브젝트의 모양 바꾸기 (예: 웃는 얼굴)
  • 모든 코드 멈추기

🎯 미션: 목적지에 닿았을 때 일어날 재미있는 성공 이벤트를 직접 상상해서 코딩해 보세요!

▶ 도움이 필요하다면! (상세 가이드 보기)

1. [시작] 탭에서 시작하기 버튼을 클릭했을 때 블록을 꺼내요.

2. [흐름] 탭의 계속 반복하기 안에 만약 참(이)라면 블록을 넣어요.

3. '참' 자리에 [판단] 탭의 목적지에 닿았는가?를 넣어요.

4. 만약 블록 안에 [생김새] 탭의 안녕! 을(를) 4초 동안 말하기를 넣고 글자를 바꿔봐요.

5. 마지막으로 [흐름] 탭의 모든 코드 멈추기를 붙여주면 완벽해요!

👑 보너스 미션 (스피드런 도전!)

미션을 모두 빠르고 완벽하게 끝냈나요? 그렇다면 아래 3가지 특별한 미션 중 하나를 골라 코드를 더 멋지게 꾸며보세요!

🖌️ 1. 발자국 남기기

엔트리의 '붓' 확장 블록을 추가해서, 주인공이 미로를 지나간 자리에 선(발자국)이 그려지게 만들어 보세요.

💎 2. 보물 획득하기

미로 중간에 보석 오브젝트를 추가하고, 주인공이 닿으면 "띠링~" 소리와 함께 보석이 사라지게 코딩해 보세요.

⏱️ 3. 초시계 마법

계산 블록의 '초시계'를 사용해서, 목적지까지 탈출하는 데 총 몇 초가 걸렸는지 화면에 기록해 보세요!

🤔 코딩이 막히나요? 마법 힌트를 열어보세요!

순차: 컴퓨터는 위에서부터 '차례대로' 실행해요. 어느 방향으로 움직일지 잘 생각해서 이동 블록을 순서대로 쌓아보세요.
반복: 2단계 퀘스트에서 앞으로 가는 블록을 계속 붙였나요? [~번 반복하기] 블록을 쓰면 코드가 훨씬 짧아져요!
선택(조건): 3단계 퀘스트에서 벽에 닿았거나 미로를 빠져나왔다면? [만약 ~참이라면] 블록을 사용하여 특별한 규칙을 만들어보세요.
