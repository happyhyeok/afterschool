알록달록 꽃송이 그리기

엔트리 오브젝트의 중심점 · 반복 · 도장찍기 · 회전을 이용해 꽃을 만들어 봅니다.

🌸 완성 작품 미리보기

꽃잎을 도장 찍고 조금씩 회전하면 알록달록한 꽃송이가 완성됩니다.

수업 목표

✅ 오브젝트의 중심점을 옮기면 회전 모양이 달라짐을 이해합니다.
✅ 반복하기로 같은 명령을 여러 번 실행합니다.
✅ 도장찍기 + 회전하기로 꽃잎을 둥글게 배치합니다.
✅ 색깔 효과를 바꾸어 알록달록한 꽃송이를 완성합니다.

오늘의 미션

분홍 꽃잎 오브젝트를 반복해서 찍고 회전시켜 예쁜 꽃송이를 완성합니다.

핵심 원리: 꽃잎 아래쪽으로 중심점을 옮기면 꽃잎이 중심을 기준으로 빙글빙글 돌며 꽃 모양이 됩니다.

1꽃잎의 중심점 이동시키기

  1. 기존 오브젝트를 삭제합니다.
  2. 오브젝트 추가하기 → 라이브러리 선택 → 식물/기타에서 분홍 꽃잎을 넣습니다.
  3. 꽃잎의 중심점 ● 을 아래쪽 끝부분으로 옮깁니다.
  4. 꽃잎 오브젝트를 장면 창 위쪽으로 이동합니다.
중요: 오브젝트는 중심점을 기준으로 회전합니다. 중심점이 가운데 있으면 제자리에서 돌고, 아래쪽에 있으면 꽃잎이 원을 그리며 돌아갑니다.

2반복 구조 만들기

블록을 아래 순서로 연결합니다.

▶ 시작하기 버튼을 클릭했을 때 └ 5번 반복하기    └ 도장찍기

반복 횟수는 처음에는 5로 설정합니다.

3회전 블록 추가하기

방향을 90°만큼 회전하기 블록을 도장찍기 아래에 연결한 뒤, 각도를 60°로 바꿉니다.

▶ 시작하기 버튼을 클릭했을 때 └ 5번 반복하기    └ 도장찍기    └ 방향을 60°만큼 회전하기

실행하면 60도 방향으로 꽃잎이 하나씩 찍힙니다. 반복 5번이면 꽃잎 5개가 만들어집니다.

4꽃잎 색깔 바꾸기

색깔 효과를 10만큼 주기 블록을 회전 블록 아래에 연결하고, 값을 1로 바꿉니다.

▶ 시작하기 버튼을 클릭했을 때 └ 5번 반복하기    └ 도장찍기    └ 방향을 60°만큼 회전하기    └ 색깔 효과를 1만큼 주기

시작하기를 눌러 꽃잎 색이 조금씩 달라지는지 확인합니다.

연습 문제 1

명령어 블록의 색깔 효과를 밝기 효과 10으로 바꾼 뒤 실행해 봅니다.

색깔 효과를 1만큼 주기 → 밝기 효과를 10만큼 주기

실행 중에는 수정할 수 없습니다. 먼저 정지하기를 누른 뒤 수정합니다.

연습 문제 2

철교놀이 조각 오브젝트를 사용하여 바람개비 모양을 만들어 봅니다.

해야 할 일설정
오브젝트 추가라이브러리 선택 → 물건 → 취미 → 철교놀이 조각
중심점 위치아래쪽 모서리에 맞추기
반복 횟수4번
회전 각도90°

🌟 도전! 나만의 스티커 만들기

꽃송이를 만든 원리를 이용하면 더 재미있는 스티커도 만들 수 있습니다.

이번에는 재미있는 모양으로 만들 오브젝트를 추가하세요.
추가한 오브젝트의 중심점을 알맞게 옮긴 후,
반복 횟수와 회전 각도를 바꾸어 나만의 스티커를 만들어 봅시다.

바꾸는 것어떤 변화가 생길까요?
반복 횟수무늬의 개수와 모양이 달라져요.
회전 각도스티커의 모양과 느낌이 달라져요.

💡 생각해 보기

같은 오브젝트라도 반복 횟수와 회전 각도를 바꾸면 전혀 다른 스티커가 만들어집니다.

컴퓨터는 같은 명령을 반복하면서 다양한 그림과 무늬를 만들 수 있습니다. 여러 값을 자유롭게 실험해 보세요.
