공식 일러스트 만들기

오늘은 캐릭터 이미지를 만들어 보겠습니다. AI로 캐릭터 이미지를 만들 때 반드시 지켜야할 사항도 함께 알아보겠습니다.

캐릭터 만들기 시작 Canva 기록장 열기 오늘 순서 보기

🚩 오늘의 개발 브리핑

오늘은 캐릭터를 정하고, Canva에서 이미지와 기록을 완성합니다.

1. 설정이름·능력·약점·성격·색 선택
2. 이미지Canva에서 1차 이미지 생성
3. 개선최종 이미지와 카드 문구 정리
4. 기록설정·수정점·선택 이유 저장

🤖 AI는 글을 보고 어떻게 그림을 만들까?

생성형 AI는 우리가 쓴 글에서 중요한 특징을 찾고, 그 특징을 바탕으로 새로운 이미지를 만들어 봅니다.

인공지능 기초
1

글로 설명하기

사람이 원하는 모습을 문장으로 씁니다.

2

특징 찾기

AI가 색, 능력, 동물, 분위기 같은 단어를 찾습니다.

3

이미지 후보 만들기

AI가 여러 모습의 그림 후보를 만들어 봅니다.

4

사람이 선택·수정하기

사람이 결과를 고르고, 필요하면 다시 요청합니다.

🧭 이런 기술은 어디에 쓰일까?

텍스트로 이미지를 만드는 기술은 게임, 웹툰, 광고, 교육 자료, 제품 디자인, 가상 공간 제작 등에 활용될 수 있습니다.

🎮 게임

캐릭터, 아이템, 배경 아이디어를 빠르게 그려 봅니다.

📚 웹툰·애니메이션

장면 분위기와 캐릭터 콘셉트를 상상해 봅니다.

📢 포스터·광고

행사 포스터나 홍보 이미지를 만듭니다.

🏫 교육 자료

수업 설명 그림이나 활동지 이미지를 준비합니다.

🧸 제품 디자인

장난감, 옷, 가방, 로봇의 모습을 미리 생각해 봅니다.

🌐 가상 공간

메타버스 배경, 아바타, 소품 아이디어를 만듭니다.

기억하기: AI는 도구이고, 무엇을 만들지 정하고 결과를 고르는 사람은 우리입니다.

🎮 1단계. 캐릭터 이름 정하기

카드를 누르거나 직접 입력해 이름을 빠르게 정합니다. 오래 고민하지 말고, 마음에 드는 조합을 선택해 보세요.

목표: 이름 확정

① 속성 카드

② 대상 카드

③ 이름 끝말 카드

내 캐릭터 이름아직 이름이 완성되지 않았어요

예: 번개 + 고양이 + 루미 → 번개냥 루미

🧩 2단계. 캐릭터 설정하기

AI가 캐릭터를 잘 그리도록 능력, 약점, 성격, 대표 색을 선택하거나 직접 입력합니다.

⚡ 능력 카드

🛡️ 약점 카드

🙂 성격 카드

🎨 대표 색 카드

이름아직 이름이 완성되지 않았어요
능력아직 선택하지 않았어요
약점아직 선택하지 않았어요
성격아직 선택하지 않았어요
대표 색아직 선택하지 않았어요

📋 3단계. AI 이미지 생성 문장 복사하기

아래 문장은 선택한 카드 내용을 바탕으로 자동 완성됩니다. 복사해서 Canva 이미지 생성창에 붙여넣으세요.

복사 1번

복사 후 Canva 기록장에서 요소 → 생성 기능을 열고 붙여넣습니다.

🖼️ 4단계. Canva에서 캐릭터 이미지 만들기

Canva 기록장에서 AI 이미지를 만들고, 가장 마음에 드는 이미지 1개를 기록장에 넣습니다.

Canva 작업

학생 활동 순서

① Canva 기록장 열기 → ② 요소 → 생성 → ③ 문장 붙여넣기 → ④ 이미지 생성 → ⑤ 이미지 1개 선택 → ⑥ 기록장에 넣기

1차 이미지를 만든 뒤 수정 요청을 추가해 최종 이미지를 한 번 더 만들어 봅니다.
다만 이미지는 2~3번까지만 사용합니다. 많이 사용하면 더 이상 만들어 주지 않아요.
