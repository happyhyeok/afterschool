🏔️ 정선 명소 타자 게임 만들기

리스트와 복제본을 활용해 정선의 명소를 맞춰봅시다!

시작 전 준비하기
1 엔트리 접속
2 로그인
3 작품 만들기
1단계: 우리 지역 명소 데이터 담기
점수: 0

핵심 개념: 리스트(List)

아우라지, 레일바이크, 화암동굴... 정선의 많은 명소들을 하나하나 따로 관리하면 힘들겠죠?
여러 개의 소중한 정보를 한 줄로 묶어 보관하는 ‘리스트’ 주머니를 사용해 봅시다.

2단계: 엔트리 실습 도전!
미션 0. 여량초 주변 배경과 오브젝트 준비
코딩 오브젝트 : 오브젝트 추가 - 배경, 단어(글상자), 안내자(엔트리봇 등)를 준비해요.
1. [배경]에서 정선의 아름다운 자연과 어울리는 숲이나 들판 배경을 선택하세요.
2. [글상자] 탭에서 내용은 비워두고 추가하세요. (여기에 정선 명소가 나타날 거예요!)
3. 문제를 내줄 안내자 오브젝트를 하나 추가하세요.
미션 1. 정선 명소 리스트 만들기
코딩 오브젝트 : 속성 탭 - '정선명소' 리스트와 '점수' 변수를 생성해요.
1. [리스트 추가]: '정선명소' 리스트를 만들고 아래 버튼을 눌러 우리 지역 단어들을 복사해 넣으세요.
2. [변수 추가]: '점수' 변수를 추가하여 성공할 때마다 점수가 올라가게 합니다.
미션 2. 무한 복제본 생성 알고리즘
코딩 오브젝트 : 글상자 - 단어들이 계속 나타나도록 복제본을 만들어요.
- 시작하기 버튼을 클릭했을 때
- 계속 반복하기 [ 2초 기다리기, 자신의 복제본 만들기 ]
미션 3. 복제본: 랜덤 정선 단어 설정
코딩 오브젝트 : 글상자 - 복제본이 정선명소 중 하나를 무작위로 선택해요.
- 복제본이 처음 생성되었을 때
- x좌표를 (-200 ~ 200) 사이의 무작위 위치로 이동
- 글상자 내용을 [정선명소 리스트의 (1 ~ 리스트 항목 수) 무작위 수]번째 항목으로 바꾸기
미션 4. 복제본: 중력 낙하 로직
코딩 오브젝트 : 글상자 - 단어가 위에서 아래로 천천히 떨어지게 해요.
- 자신의y좌표가 -150보다 작아질 때까지 반복하기
- [ y좌표를 -3만큼 바꾸기 ]
- 바닥에 닿으면 [ 이 복제본 삭제하기 ]
미션 5. 안내자: 입력 창 띄우기
코딩 오브젝트 : 안내자 - 우리가 타자를 칠 수 있게 계속 물어봐요.
- 시작하기 버튼을 클릭했을 때
- 계속 반복하기 [ "정선의 명소는?" 라고 묻고 대답 기다리기 ]
미션 6. 복제본: 정답 판정 및 득점
코딩 오브젝트 : 글상자 - 내가 입력한 글자와 단어가 같으면 점수를 줘요.
- 미션 4의 낙하 반복문 안에 다음 조건문을 넣으세요:
- 만일 [ 대답 = 자신의 글상자 내용 ] 이라면
- [ '점수'에 10만큼 더하기 ]
- [ 이 복제본 삭제하기 ]
