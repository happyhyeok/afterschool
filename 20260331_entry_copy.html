복제의 이해

하늘에서 음식이 내린다면? 🍔

마법의 '복제' 블록으로 음식을 쏙쏙 받아먹는 게임을 만들어 봐요!

💻

1. 코딩 마법을 부릴 준비!

1단계: 엔트리 접속

크롬 브라우저를 열고
'엔트리' 사이트에 들어가요.

2단계: 로그인

오른쪽 위 버튼을 눌러
내 아이디로 로그인해요.

3단계: 작품 만들기

상단 메뉴에서 [만들기] ➔
[작품 만들기]를 클릭해요.

🎒

2. 필요한 오브젝트 꺼내기

💡 기존에 있던 '엔트리봇'은 지우고, 아래 3가지 오브젝트를 추가해 주세요!

🍕
음식 (위에서 떨어질 아이템)

피자, 햄버거, 도넛, 사과 등 내가 가장 좋아하는 음식을 골라요.

🏃‍♂️
캐릭터 (아래에서 받아먹을 주인공)

음식을 요리조리 피하며 받아먹을 귀여운 동물이나 사람을 골라요.

🏞️
배경 (게임 무대)

식당, 우주, 들판 등 게임에 어울리는 멋진 배경을 골라주세요.

🪄

3. 마법의 '복제' 블록 파헤치기

음식을 100개 내리게 하려고 음식을 100번 추가할 필요가 없어요!
진짜 음식(원본)은 숨어서 공장장 역할을 하고, 마법처럼 가짜 음식(복제본)을 계속 만들어 낼 거예요.

1

진짜 나(원본)가 나와 똑같이 생긴 쌍둥이를 화면에 '뿅!' 하고 만들어내는 주문이에요. 이 블록을 계속 반복하면 복제본이 무한대로 생겨나요!

2

새로 태어난 복제본(가짜 음식)들에게 "너희들은 이제 위에서 아래로 떨어져!"라고 명령을 내리는 시작 블록이에요. 복제본들은 이 블록 아래에 연결된 명령만 들어요.

(매우 중요!)

음식을 먹었거나 음식이 땅에 떨어졌다면? 화면에서 지워줘야 해요!
다 쓴 복제본을 삭제하지 않고 쌓아두면 컴퓨터가 쓰레기로 가득 차서 게임이 엄청 느려지고 멈춰버린답니다.

⚙️

4. 오브젝트별 코딩 가이드

아래 안내를 보고 각 오브젝트를 선택한 후, 알맞은 블록을 조립해 보세요!

🏃‍♂️ 캐릭터 (받아먹는 친구)

① 키보드의 왼쪽, 오른쪽 화살표를 눌러 캐릭터가 왼쪽, 오른쪽으로 움직이게 만들어요.

[시작하기 버튼을 클릭했을 때] 계속 반복하기 만약 [오른쪽 화살표] 키가 눌렸는가? 이라면 ➔ x 좌표를 10 만큼 바꾸기 만약 [왼쪽 화살표] 키가 눌렸는가? 이라면 ➔ x 좌표를 -10 만큼 바꾸기

🍕 진짜 음식 (숨은 공장장)

① 음식이 계속 자신의 복제본을 만들어요.
진짜 음식은 게임 화면에 보이지 않아요. 숨어서 0.5초마다 가짜 음식(복제본)을 찍어내기만 합니다.

[시작하기 버튼을 클릭했을 때] ➔ 모양 숨기기 // 화면에서 사라져서 공장장이 됩니다! 계속 반복하기 ➔ 자신의 복제본 만들기 ➔ 0.5 초 기다리기

✨ 복제된 음식 (하늘에서 떨어지기)

① 복제된 음식들이 위에서 아래로 떨어지게 만들어요.
② 떨어지다가 캐릭터에 닿으면 삭제되게 만들어요.
③ 떨어지다가 바닥에 닿으면 삭제되게 만들어요.

[복제본이 처음 생성되었을 때] ➔ 모양 보이기 ➔ x위치: [-200] ~ [200] 사이의 무작위 수, y위치: [150] 으로 이동하기 계속 반복하기 ➔ y 좌표를 -5 만큼 바꾸기 // 아래로 떨어집니다. 만약 [캐릭터] 에 닿았는가? 이라면 // 냠냠 먹었을 때 ➔ [먹는 소리] 재생하기 ➔ 이 복제본 삭제하기 만약 [아래쪽 벽] 에 닿았는가? 이라면 // 바닥에 떨어졌을 때 ➔ 이 복제본 삭제하기
🎮

5. 딩동댕 퀴즈!

Q. 코딩을 할 때 다 쓴 복제본을 '이 복제본 삭제하기' 블록으로 지워주지 않으면 어떤 일이 생길까요?

🚀

6. 보너스 미션: 게임 업그레이드!

시간이 남았다면, 아래 3가지 미션에 도전해서 나만의 멋진 게임으로 만들어보세요!

💯

미션 1: 점수판 만들기

자료 탭에서 '점수' 변수를 만들어요. 음식을 먹을 때마다(닿았을 때) 점수가 10점씩 올라가게 코딩해 보세요!

🎨

미션 2: 다양한 음식 내리기

음식 오브젝트의 '모양' 탭에 여러 음식을 추가해요. 복제본이 처음 생성될 때 모양을 무작위로 바꿔보세요!

💣

미션 3: 폭탄 피하기

새로운 오브젝트 '폭탄'을 추가하고 똑같이 복제 코드를 넣어요. 폭탄을 먹으면 점수가 깎이거나 게임이 끝나게 만들어 보세요!
