1 오브젝트와 시작 위치

① 4개의 오브젝트를 추가하세요.

  • • 👤 주인공 (나)
  • • 🚧 장애물
  • • 🚩 목표 지점
  • • 🏞️ 게임 배경

② 시작 위치 고정 알고리즘

[모든 오브젝트 공통] ① 시작하기 버튼을 클릭했을 때 ② 지정된 [x: ( ), y: ( )] 위치로 이동하기
📍 추천 좌표 (시작할 때 위치)
  • • 주인공: x: 0, y: -130
  • • 목표: x: 0, y: 130
  • • 장애물: x: -200, y: 0

힌트 이미지:

오브젝트를 선택한 후 이 블록을 연결해 주세요.

2 [주인공] 조종 알고리즘

키보드 화살표를 눌러 캐릭터를 조종해 봅시다.

[주인공 오브젝트] ① 시작하기 버튼을 클릭했을 때 ② 계속 반복해서 ②-1 만일 위쪽 화살표 키를 눌렀다면 └ y 좌표를 10만큼 바꾸기 ②-2 만일 아래쪽 화살표 키를 눌렀다면 └ y 좌표를 -10만큼 바꾸기

힌트 이미지:

'무한 반복하기' 안에 '만일 ~라면' 블록을 겹쳐서 사용합니다.

3 [장애물] 무한 이동 알고리즘

장애물이 왼쪽에서 오른쪽으로 계속 지나가게 합니다.

[장애물 오브젝트] ① 시작하기 버튼을 클릭했을 때 ② 계속 반복해서 ②-1 x 좌표를 5만큼 바꾸기 ②-2 만일 오른쪽 벽에 닿으면 (x 좌표 > 240) └ x 위치를 -240으로 이동하기

힌트 이미지:

오른쪽 끝(240)을 넘어가면 다시 왼쪽 끝(-240)으로 보내는 논리입니다.

4 [목표] 도착 성공 판정

목표 지점에 주인공이 닿으면 게임이 끝납니다.

[목표 오브젝트] ① 시작하기 버튼을 클릭했을 때 ② 계속 반복해서 ②-1 만일 주인공 에 닿았는가? 라면 └ "성공!"을 2초 동안 말하기 └ 모든 코드 멈추기

힌트 이미지:

성공 메시지를 말한 뒤 '모든 코드 멈추기'로 게임을 종료합니다.

🚀 심화 도전 미션
5단계: 충돌 시 초기화

주인공 코드 안에 '만일 장애물에 닿았는가?'를 추가하고, 다시 시작 위치(x:0, y:-130)로 보내는 코드를 짜보세요.

6단계: 속도 변화 주기

장애물 코드에서 x 좌표를 바꾸는 값을 10번 반복하기 블록을 두 번 사용하여 빠르다가 느려지게 만들어보세요.
