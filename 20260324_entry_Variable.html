📦

📦 2단계: 변수 블록 이해하기

엔트리에는 변수 값을 바꾸는 중요한 마법 블록들이 있어요. 버튼을 눌러 상자 안의 숫자가 어떻게 변하는지 확인해 보세요!

📦
점수 상자
0
🎯 정하기 마법

상자 안의 내용을 다 버리고 새로운 숫자 하나만 쏙 넣는 마법이에요. 보통 게임 시작할 때 '0'으로 만들 때 써요!

➕ 더하기 마법

원래 들어있던 숫자에 새로운 숫자를 합치는 마법이에요. 보물을 먹을 때마다 점수가 올라가게 할 때 써요!

💡 점수 값 블록은 언제 쓰나요?

상자 안에 들어있는 진짜 숫자를 알고 싶을 때 사용해요!
예를 들어 "내 점수는 (점수 값)점이야!"라고 말하게 하거나, "점수가 10점인가?"라고 비교할 때 이 블록을 가져다 써야 해요.

🛠️

🛠️ 3단계: '점수' 변수 만들기

엔트리 [속성] ➔ [변수] ➔ [+ 변수 추가하기]에서 이름을 '점수'로 입력하세요.

💎

💎 4단계: 보물 코딩하기

1

보물 오브젝트 추가

미로 여기저기에 동전, 보석, 사과 오브젝트를 추가하세요.

2

점수 획득 코드 작성

🚩 오늘의 핵심 미션

• 코딩할 오브젝트 : 동전, 보석, 사과

• 코딩할 내용 : 주인공이 보물(동전, 보석, 사과)을 먹으면 점수가 1씩 더해지게 만들기

▶ 코딩 블록 힌트 보기
3

테스트하기

보물을 먹었을 때 점수가 정확히 1씩 올라가는지 확인해 보세요.

🚀 5단계: 레벨업 미션

🥇

미션 1: 다른 점수 주기

사과는 5점, 동전은 10점, 보석은 100점을 주게 고쳐보세요.

👻

미션 2: 감점 장애물 만들기

닿으면 점수가 -2점씩 깎이는 '무서운 유령'을 넣어보세요.

📢

미션 3: 내 점수 자랑하기

목적지에 도착하면 "내 최종 점수는 (점수값)점이야!"라고 자랑하게 만들어 보세요.
