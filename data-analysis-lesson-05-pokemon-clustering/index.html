ENTRY 1

CSV 데이터 추가하기

먼저 AI가 공부할 데이터를 엔트리에 넣어야 합니다.

  1. 엔트리 작품을 엽니다.
  2. 왼쪽 메뉴에서 [데이터 분석] 또는 [테이블]을 찾습니다.
  3. [테이블 추가하기]를 클릭합니다.
  4. 오늘 받은 CSV 파일을 업로드합니다.
  5. 표에 캐릭터명, 영문명, 공격력, 귀여움지수가 보이는지 확인합니다.

ENTRY AI

군집 모델 만들기

이제 AI가 캐릭터들을 비슷한 친구끼리 묶도록 모델을 만들어 봅시다.

왜 K=3으로 할까요?

처음부터 너무 많은 그룹을 만들면 해석하기 어렵습니다. 오늘은 결과를 쉽게 비교하기 위해 3개 그룹으로 나누어 봅니다.

  • 귀여움지수가 높은 캐릭터
  • 공격력과 귀여움지수가 중간인 캐릭터
  • 공격력이 높은 캐릭터
설정 항목 선택할 값
모델 종류 군집: 숫자
알고리즘 K-평균
모델 이름 포켓몬_군집모델
핵심 속성 공격력, 귀여움지수
군집 개수 3
  1. [인공지능] 메뉴를 클릭합니다.
  2. [인공지능 모델 학습하기]를 클릭합니다.
  3. [새로 만들기]를 선택합니다.
  4. [군집: 숫자]를 선택합니다.
  5. 모델 이름을 ‘포켓몬_군집모델’로 입력합니다.
  6. 데이터 테이블을 선택합니다.
  7. 핵심 속성으로 ‘공격력’과 ‘귀여움지수’를 선택합니다.
  8. 군집 개수를 3으로 설정합니다.
  9. [학습하기]를 클릭합니다.
  10. 결과 차트에서 점들이 3개 그룹으로 나뉘는지 확인합니다.

가까운 데이터를 찾아 자동으로 그룹화합니다.
